## Install:saxophone:

```sh

```

## Usage

```sh
npm run serverstart
```

![](\Web server form handling.png)

- `catalog/` ‚Äî The home/index page.
- `catalog//` ‚Äî The list of all books, bookinstances, genres, or authors (e.g. /`catalog/books/`, /`catalog/genres/`, etc.)
- `catalog//**` ‚Äî The detail page for a specific book, bookinstance, genre, or author with the given `*_id*` field value (e.g. `/catalog/book/584493c1f4887f06c0e67d37)`.
- `catalog//create` ‚Äî The form to create a new book, bookinstance, genre, or author (e.g. `/catalog/book/create)`.
- `catalog//**/update` ‚Äî The form to update a specific book, bookinstance, genre, or author with the given `*_id*` field value (e.g. `/catalog/book/584493c1f4887f06c0e67d37/update)`.
- `catalog//**/delete` ‚Äî The form to delete a specific book, bookinstance, genre, author with the given `*_id*` field value (e.g. `/catalog/book/584493c1f4887f06c0e67d37/delete)`.
